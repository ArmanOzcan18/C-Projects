Fall 2022 Final Exam: Problem 4
-------------------------------

Running
-------

To build everything, just run "make".

Then, run the command shown in the problem.


Exploring
---------

Try changing the arguments you give to fgecho.  Note that if you give any
argument longer than 8 characters, you will cause a memory error.

A good exercise is to fix fgecho to avoid these memory errors!  (It's similar to
what you did in Lab 2 part 2.)
