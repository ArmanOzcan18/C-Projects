Fall 2022 Final Exam
--------------------

This repository contains the COMS W3157 final exam from Fall 2022, along with
its solutions, explanations, and accompanying code.

The exam, its solutions, and explanations can be found in:

    3157-final-2022-fall.txt
    3157-final-2022-fall-answers.txt
    3157-final-2022-fall-explanations.txt

The code for problem N can be found in the directory named qN/; e.g., the code
for problem 5's is in q5/.  In each directory, there is a README.txt explaining
what to do in that directory.
